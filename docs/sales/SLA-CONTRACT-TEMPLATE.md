# قالب SLA — عمران‌سام

## سطح Bronze
- uptime هدف: ۹۹٪
- بک‌آپ: هفتگی
- پاسخگویی: ۴۸ ساعت کاری
- شامل: هاست، SSL، رفع باگ

## سطح Silver
- uptime هدف: ۹۹.۵٪
- بک‌آپ: روزانه
- پاسخگویی: ۲۴ ساعت کاری
- شامل Bronze + ۵ ساعت edit/ماه + patch امنیتی

## سطح Gold
- uptime هدف: ۹۹.۹٪
- پاسخگویی: ۴ ساعت (ساعات اداری)
- شامل Silver + ۱۵ ساعت edit + SMS + پشتیبانی تلفنی

## خارج از SLA
- توسعه ماژول جدید: برآورد جدا
- ساعت اضافه: ۸۰۰ هزار – ۱.۲ میلیون/ساعت
