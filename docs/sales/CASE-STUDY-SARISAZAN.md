# Case Study — سامانه مدیریت پروژه عمرانی

## مشتری
شرکت پیمانکاری عمرانی (نمونه: سریع‌سازان البرز)

## چالش
- Excel و فایل‌های پراکنده
- نبود گزارش لحظه‌ای
- پیگیری سخت صورت‌وضیت و حضور میدانی

## راه‌حل Omransam Core
- داشبورد KPI
- مدیریت پروژه و قرارداد
- صورت‌وضعیت و مالی
- مناقصات، PWA میدانی، دبیرخانه
- تقویم جلالی، deploy اختصاصی

## نتایج
- یکپارچگی داده
- کاهش خطا در مالی
- گزارش سریع‌تر برای مدیریت
- دسترسی موبایل در محل پروژه

*برای انتشار عمومی، تأیید کتبی مشتری لازم است.*
